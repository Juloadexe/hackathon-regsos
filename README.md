```
┌─────────────────┐    HTTP запрос    ┌──────────────────┐    Чтение файла    ┌─────────────────┐

│                 │    JSON ответ     │                  │   Парсинг данных   │                 │
└─────────────────┘                   └──────────────────┘                    └─────────────────┘
         │                                      │                                      │
         │                                      │                                      │
         │ Показывает красивые                  │ Отправляет структурированные         │ Используется как
         │ таблицы и графики                    │ данные и статистику                  │ исходный материал
         ▼                                      ▼                                      ▼
```
